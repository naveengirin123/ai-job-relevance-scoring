# AI Job Relevance Scoring Engine

## Overview
This project is an AI-assisted Python tool that automates job filtering and scoring based on relevance to a candidate's skills.

The system can:
- Parse job descriptions
- Score jobs based on skills relevance
- Filter relevant opportunities
- Automate job search workflows

## Project Structure

scoring_engine.py → scoring logic  
job_processor.py → job filtering  
naukri_agent.py → automation agent  
report_generator.py → reporting  
scheduler.py → automation timing  
section_parser.py → text parsing  

## Installation

Install dependencies:

pip install -r requirements.txt

## Usage

Run:

python naukri_agent.py
